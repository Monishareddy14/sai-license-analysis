# sai-license-analysis
# Sai Excel Analysis (Pandas)

## What this project does
1. Calculates active users per month and department using StartDate and EndDate.
2. Calculates login counts per user by Year/Month and buckets them (0, 01-10, 11-20, >=21).
3. Calculates license availability per month from License Procurement sheet.

## Tools used
- Python
- Pandas
- Excel

## How to run
1. Install packages: `pip install -r requirements.txt`
2. Update Excel path in code
3. Run the notebook/script
